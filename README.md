# 🛒 E-Commerce Sales Analysis

## 📌 Project Overview
Analysis of 9,800 orders from a global superstore
using Python and Power BI (2015-2018).

## 🛠️ Tools Used
- Python (Pandas, Matplotlib)
- Power BI Dashboard
- VS Code
- Kaggle Dataset

## 📊 Key Analyses Done
- Yearly Sales Performance
- Region Wise Sales
- Category Analysis
- Sub-Category Analysis
- Top 10 Customers
- Monthly Sales Trend

## 💡 Key Insights
- 2018 was best year — $722,052 sales
- West region leads — $710,219
- Technology top category — 36.6% share
- November peak month — $351,426
- Sean Miller top customer — $25,043
